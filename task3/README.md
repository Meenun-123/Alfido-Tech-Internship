This project is a creative background color changer built for the Alfido Tech Internship Task 3. Clicking the button generates a random color, smoothly updates the background, adjusts text color automatically, and tracks how many colors have been visited.

## Features
- Smooth background color transition  
- Automatic text color contrast detection  
- Animated gradient button  
- Fade transition for button gradient  
- Displays current color  
- Tracks how many colors were visited  
- Shows total possible hex colors (16,777,216)

## File Structure
```
index.html
style.css
script.js
```

## How to Use
Open `index.html` in any browser and click **Change Background**.
